Readme:1. to run this program, you can use command line: java Convexhull
   to compile this program, you can use command line: javac -Xlint Convexhull.java
2. In this program, int n which is default 100 is the number of points. This program will randomly create n points. If you want to change the n value, you can change it in the begin of the program.
3. In the output of this program, it will print the n points, the convex hull points, and the time which the program used to get the result.
