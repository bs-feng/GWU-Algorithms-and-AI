Readme:1. to run this program, you can use command line: java Magicaleggs
   to compile this program, you can use command line: javac Magicaleggs.java
2. In this program, the number of eggs and floors are given in the beginning of the program. If you want to change the value, you can change it in the begin of the program.
3. In the output of this program, it will print the number of throws, the floor from which to drop the first egg and the processing time of the program.
